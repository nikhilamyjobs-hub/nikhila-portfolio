## 🚢 Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm install -g vercel
vercel login
vercel --prod
```

### Option 2 — GitHub + Vercel (recommended)
1. Push this repo to GitHub
2. Go to vercel.com → New Project
3. Import your GitHub repo
4. Click Deploy — done ✅
